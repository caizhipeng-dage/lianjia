# 厦门二手房数据采集、分析、挖掘
使用 Python 的爬虫工具，对厦门链家网在售和成交的二手房源进行数据采集；对采集的 26500 条结果进行数据处理，分析现阶段厦门各区域二手房价格和交易情况；最后采用机器学习的决策树回归模型，搭建房价预测模型，预测厦门二手房价格。
